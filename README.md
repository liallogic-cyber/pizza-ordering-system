# Pizza Ordering System 🍕

## 1. Project Overview

A terminal-based Java application that simulates pizza ordering. It demonstrates the integration of JDBC for order persistence and the application of multiple design patterns.

## 2. Core Features

* Users can select pizzas from a menu, choose sizes, and add toppings.
* The system calculates the final order total.
* Orders are saved to a PostgreSQL database using JDBC.

## 3. Technologies Used

* **Language:** Java 17
* **Database:** PostgreSQL
* **Library:** JDBC (Java Database Connectivity)
* **Patterns:** DAO, Decorator, Builder, Strategy

## 4. How to Run the Project

### Prerequisites

* Java JDK 17+ installed
* PostgreSQL installed and running

### Setup

1. Clone the repository: `git clone https://github.com/liallogic-cyber/pizza-ordering-system`
2. Navigate to the project directory: `cd pizza-ordering-system`
3. Set up the database by running the script located in `database/schema.sql`
4. Compile the code: `javac src/Main.java`
5. Run the application: `java src/Main`
